COMO CRIAR UMA NOVA LISTA DE COMPRAS

  1. Passo 1: Clique no botão “Criar Nova Lista” na página inicial.
  2. Passo 2: Dê um nome à sua lista, como “Supermercado” ou “Itens para casa”.
  3. Passo 3: Informe os itens que deseja comprar. Você pode adicioná-los manualmente ou usar a busca para encontrar itens populares.
  4. Passo 4: Organize os itens por categoria (como alimentos, produtos de limpeza, etc.) para facilitar na hora de encontrar no mercado.

    5. Dica: Você pode adicionar ou remover itens, personalizar e editar a lista a qualquer momento.

COMO VER MINHAS COMPRAS ANTERIORES

  1. Passo 5: Acesse a seção “Histórico de Compras” no menu principal.
  2. Passo 6: Você verá uma lista de todas as compras feitas anteriormente, organizadas por data.
  3. Passo 7: Clique em qualquer lista para ver os itens que foram comprados, quantidades e valores.

COMO ACOMPANHAR MINHAS DESPESAS

  1. Passo 8: Vá para a seção “Minhas Despesas”.
  2. Passo 9: Você verá um resumo de seus gastos com categorias, como alimentos, bebidas, etc.
  3. Passo 10: Gráficos e tabelas mostram seu consumo mensal.

    4. Dica: Use a visão semanal ou mensal para obter melhores dados sobre seus hábitos de compra.

  4. Passo 11: Clique em itens ou categorias para ver detalhes.
  1. Passo 12: Para evitar esquecimentos, você pode ativar lembretes de hora ou dia específicos.
  2. Passo 13: Suas listas e despesas ficam sincronizadas com a conta, acessíveis de qualquer dispositivo.

    3. Dica: Sempre que possível, utilize categorias e preços cada compra para manter tudo organizado.
